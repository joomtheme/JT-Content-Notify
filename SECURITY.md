# Security Policy

## Supported versions

Security fixes are provided for the latest published release of JT Content Notify.

| Version | Supported |
|---|---|
| Latest release | Yes |
| Older releases | Upgrade recommended |

## Reporting a vulnerability

Do not open a public GitHub issue for a suspected security vulnerability.

Report security issues privately to:

**support@joomtheme.com**

Please include:

- JT Content Notify version
- Joomla and PHP versions
- Affected component or module
- Detailed reproduction steps
- Expected security impact
- Relevant logs or proof of concept
- Whether the issue is already publicly known

Do not include real user data, production credentials, or sensitive tokens.

JoomTheme will review the report, confirm whether it is reproducible, and coordinate a fix and disclosure when appropriate.
